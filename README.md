## 🧠 AI-Powered Facial Expression Analysis

**Sentiment Vision AI** is a real-time emotion detection web app that analyzes facial expressions using your webcam. Powered by TensorFlow.js and FaceMesh, it detects emotions instantly - all in your browser with zero privacy concerns!

### ✨ Features

- **Real-time Emotion Detection**: Happy, Sad, Surprised, Angry, Neutral
- **Live Webcam Integration**: Works instantly with your camera
- **Facial Landmark Visualization**: See AI tracking your face
- **Emotion Confidence Scores**: Percentage breakdown for each emotion
- **Snapshot Feature**: Save your emotion analysis as an image
- **100% Private**: All processing happens locally - no data uploads

### 🛠️ Technology Stack

- TensorFlow.js
- FaceMesh Model (Google MediaPipe)
- HTML5 Canvas & WebRTC
- CSS3 Animations

### 📸 How It Works

1. Click "Start Camera" and allow webcam access
2. AI detects your face and facial landmarks
3. Real-time emotion analysis begins
4. See instant feedback on your emotional state
5. Take snapshots to save your analysis
